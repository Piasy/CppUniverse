# Djinni Hello World

A hello world project of [djinni](https://github.com/dropbox/djinni).

## CPP Project

+ `cd cpp_project`;
+ `xcake make`;
+ Open `HelloWorld.xcodeproj` in Xcode;

## Android Project

+ Just open the `android_project` folder in Android Studio.

## iOS Project

+ `cd ios_project`;
+ `xcake make`;
+ Open `HelloWorld.xcodeproj` in Xcode;
